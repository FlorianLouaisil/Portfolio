Cook'N'Co

Lien GitHub

Description

Cook'NCO est une application web permettant de recenser des recettes de cuisine. Chaque recette contient des informations détaillées telles que le temps de préparation, la liste des ingrédients avec leurs quantités et les étapes de préparation. L’objectif est d’offrir une interface intuitive et pratique pour les passionnés de cuisine.

Problématique rencontrée

Dans la première version du frontend, toutes les recettes étaient chargées simultanément, ce qui pouvait entraîner des problèmes de performance à mesure que leur nombre augmentait. La récupération des données a été optimisée en adaptant le mécanisme de chargement des recettes, afin d'éviter une surcharge et d'améliorer ainsi les performances de l'application.

Compétences développées

  • Gestion des composants React
  • Optimisation des appels API et gestion des états de chargement et d'erreur

API Cook'n'Co

Lien GitHub

Description

API développée avec Express pour gérer les recettes de l’application Cook'NCO. Elle permet la gestion des données liées aux recettes (ajout, modification, suppression), ainsi que l’interaction avec une base de données MySQL pour stocker les informations sur les recettes, les ingrédients et leur préparation.

Problématique rencontrée

Au départ, j’ai utilisé MongoDB pour la gestion des données. Cependant, après réflexion, j'ai constaté que ce choix n’était pas optimal pour ce projet en raison de la structure relationnelle des données (recettes, ingrédients, etc.). J'ai donc décidé de passer à MySQL afin de mieux gérer les relations entre les données et d’améliorer les performances des requêtes.

Compétences développées

  • Développement d’une API REST avec Express
  • Gestion et sécurisation des données
  • Optimisation des requêtes SQL

Optimisation SEO et Débogage

Lien GitHub

Description

Dans le cadre de ma formation, j’ai été chargé d’optimiser le site d’une photographe pour améliorer sa visibilité sur les moteurs de recherche (SEO) et réduire les temps de chargement. Cela a impliqué l’optimisation des images (réduction de leur taille sans perte de qualité), ainsi que la réorganisation et l’optimisation du code pour favoriser une meilleure expérience utilisateur et un meilleur référencement. J’ai aussi amélioré l'accessibilité en ajoutant des balises sémantiques au code.

Problématique rencontrée

Le site de la photographe était relativement lent à charger, principalement à cause des images trop lourdes et d’une structure de code non optimisée pour le référencement. J’ai donc réalisé un audit complet du site, en utilisant des outils comme Lighthouse pour mesurer les performances et Wave pour l’accessibilité. Puis, j’ai mis en œuvre des solutions pour optimiser les images, ajouter des balises sémantiques et améliorer la structure du code afin d’optimiser la vitesse de chargement et d'améliorer le référencement du site.

Compétences développées

  • Optimisation du temps de chargement
  • Amélioration du référencement SEO
  • Amélioration de l'accessibilité
  • Utilisation d’outils d’audit tels que Lighthouse et Wave
